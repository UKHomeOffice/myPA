Proposal


A simple tool  to aggregate the  fragmented on-boarding and off-boarding processes into a  workflow.


The suggested  tool will not replace or  require any changes to exiting Home Office systems and procedures but it  will simply;


1. automate form submissions

2. handle sending and receiving emails

3. provides users with a dashboard to track on-boarding and off-boarding  requests


User will supply info required to complete an on-boarding / off-boarding transaction only once.


The tool will automate everything , from form filling to tracking progress.


The tool works like a PA, so we are calling it myPA
